# Waitkit GitHub Review Reply Formats

Use concise Korean replies.

## Fixed

```text
반영했습니다. 지적해주신 부분을 수정했고 관련 검증도 다시 확인했습니다.
```

## Not Applied

```text
확인해보니 현재 Waitkit 규칙과 실제 동작 기준에서는 변경하지 않는 편이 맞다고 판단했습니다. 이유는 다음과 같습니다: <reason>
```

## Needs Clarification

```text
의도는 이해했지만 적용 범위가 애매해서 확인이 필요합니다. <question>
```
